# Part 1: First Principles

Part 1 builds the mental model needed before joining DN42.

By the end of this part, the reader should be able to:

- Predict which interface a packet will leave.
- Explain basic source address selection.
- Distinguish an underlay path from an overlay tunnel.
- Read route tables and identify connected, default, blackhole, and learned routes.
- Explain why BGP exists before configuring it.

The first labs should be local and disposable. Public DN42 state comes later.
