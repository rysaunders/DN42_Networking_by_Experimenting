## Create AS-shaped namespaces

$ ip netns add toy-as1

$ ip netns add toy-as2

$ ip netns add toy-as3

$ ip netns add toy-as4

$ ip netns list
toy-as4
toy-as3
toy-as2
toy-as1

## Create veth links

$ ip link add as1-as2 type veth peer name as2-as1

$ ip link set as1-as2 netns toy-as1

$ ip link set as2-as1 netns toy-as2

$ ip link add as2-as3 type veth peer name as3-as2

$ ip link set as2-as3 netns toy-as2

$ ip link set as3-as2 netns toy-as3

$ ip link add as3-as4 type veth peer name as4-as3

$ ip link set as3-as4 netns toy-as3

$ ip link set as4-as3 netns toy-as4

$ ip link add as4-as1 type veth peer name as1-as4

$ ip link set as4-as1 netns toy-as4

$ ip link set as1-as4 netns toy-as1

## Configure loopback service addresses

$ ip -n toy-as1 addr add 172.20.1.1/32 dev lo

$ ip -n toy-as2 addr add 172.20.2.1/32 dev lo

$ ip -n toy-as3 addr add 172.20.3.1/32 dev lo

$ ip -n toy-as4 addr add 172.20.4.1/32 dev lo

## Configure point-to-point link addresses

$ ip -n toy-as1 addr add 10.42.12.1/30 dev as1-as2

$ ip -n toy-as2 addr add 10.42.12.2/30 dev as2-as1

$ ip -n toy-as2 addr add 10.42.23.1/30 dev as2-as3

$ ip -n toy-as3 addr add 10.42.23.2/30 dev as3-as2

$ ip -n toy-as3 addr add 10.42.34.1/30 dev as3-as4

$ ip -n toy-as4 addr add 10.42.34.2/30 dev as4-as3

$ ip -n toy-as4 addr add 10.42.41.1/30 dev as4-as1

$ ip -n toy-as1 addr add 10.42.41.2/30 dev as1-as4
